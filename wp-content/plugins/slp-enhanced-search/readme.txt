A premium add-on pack for Store Locator Plus that adds advanced search form modifications.

== Description ==

Get more control and customization of the Store Locator Plus search form.

= Features =

* Hide The Search Form
* Discrete state search
* Discrete country search

== Installation ==

= Requirements =

* Store Locator Plus: 4.6.5
* WordPress: 3.8+
* PHP: 5.2.4+

== Frequently Asked Questions ==

= Why "Legacy" =

This product will not be updated for new features only security patches.
It may not work with future versions of the base Store Locator Plus plugin.
Make sure you keep a copy of your current version of Store Locator Plus that is working with this add  on.
You may not be able to upgrade to future versions of the base plugin.

= Is There A New Version =

The new product is the [Experience](https://wordpress.storelocatorplus.com/product/experience/) Add on.
